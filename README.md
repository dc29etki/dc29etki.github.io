# LeaseTrack Community Dashboard
New Front-end Code

## Project setup

### Requirements
- Node.js
- NPM

### Install

```
$ git clone https://github.com/dc29etki/dc29etki.github.io
$ cd dc29etki.github.io
```

### Build Node modules
```
npm install
```

### Compiles and serves on Windows
```
npm run start
```

### Compiles and serves on Unix systems
```
npm run start-unix
```
